## 👨‍💻 Sobre Mí

Soy un **Desarrollador Full Stack especializado en el ecosistema .NET**, con **más de 5 años de trayectoria** creando aplicaciones empresariales robustas. Mi experiencia incluye **más de un año de enfoque intensivo en .NET MAUI**, liderando la transición hacia soluciones multiplataforma modernas.

Me apasiona construir software de **alta calidad, limpio, escalable y centrado en la experiencia del usuario**. Mis habilidades abarcan todo el ciclo de vida del desarrollo, desde la arquitectura y la base de datos hasta las interfaces de usuario web y móviles, integrando **APIs REST y SOAP** de manera segura y eficiente.

Además de mi faceta como desarrollador, encuentro una gran satisfacción como **Instructor de Programación**, compartiendo conocimientos y guiando a futuros desarrolladores.
